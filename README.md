# Client Portal
A multi-tenant client portal with a secure custom auth (Argon2 password hashing, OTP verification, session management via Redis, and password reset flow). Features real-time messaging over WebSockets, file storage with S3 compatible object storage, and a polished Next.js dashboard. Built with a single-schema multi-tenant PostgreSQL database, tested with Jest, containerized with Docker, and deployed to a DigitalOcean VPS.
